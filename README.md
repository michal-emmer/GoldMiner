# Gold Miner

Školský projekt v jazyku JavaScript, vytvorený v roku 2020.

## Popis projektu
Jednoduchá hra Gold Miner, kde hráč ťaží zlato a predmety pomocou háku v 3 leveloch.

## Ako spustiť hru
1. Skopírujte alebo stiahnite celý projekt do počítača.  
2. Otvorte súbor `GoldMiner.html` vo vašom webovom prehliadači (Chrome, Firefox, Edge…).  
3. Hra sa automaticky spustí a môžete začať hrať.

## Ovládanie
Pre spustenie háku použite klávesu šípka dole.
